# My Neovim Config
This is my Neovim config I use to code!
Feel free to use this for your own Neovim setup!

## This includes
- Completion Engine 
- One Dark Theme
- Fuzzy Finder (Telescope)
- LSP Setup for:
 1. Go (gopls) 
 1. TypeScript (tsserver)
 1. HTML (html)
 1. JSON (jsonls)
 1. Svelte (svelte)
 1. Lua (lua_ls, OFC!)
